# danmaku

“弾幕 (ダンマク) / Bullet-Hell” スタイルのシンプルシューティングゲーム  の Java 実装プロジェクトです。  

スクリーン上でプレイヤーと障害物 (あるいは敵からの「弾」) を扱い、弾を避けたり当たり判定を行ったりする、といった基本的な “弾幕ゲーム” の要素を備えています。

---

## 🎯 目的 / コンセプト  

- いわゆる “bullet-hell / danmaku” 形式のゲーム体験を Java + GUI (もしくは Java 標準の描画手段) で実現。  
- リアルタイムの描画、当たり判定、ステージ進行、スコア管理など、弾幕ゲームの基礎構造をシンプルに実装。  
- 学習用・趣味用・軽めの作品として、設定やコード構造の拡張に対応しやすいよう設計。
